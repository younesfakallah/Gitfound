Gitfound c'est quoi ?

Ce site peut s'avérer utile pour les recruteurs tech qui recherchent à analyser les travaux Github de leurs futurs talents. En spécifiant un nom d'utilisateur github à gitfound ce dernier vous renverra le plus gros projet public que cet utilisateur aura faits. Cela vous permettra de jeter un regard sur un projet suffisamment important pour vous permettre de juger le travail de cet utilisateur de façon optimale.
